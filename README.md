# CommunityPulse AI 🚨

An AI-powered emergency response and crisis prediction platform built with Gemini 2.0 Flash and React.

## 🚀 Key Features
- **Predictive Crisis Detection**: AI-powered 48h risk forecasting.
- **Live Ingestion Feed**: Real-time distress signal processing.
- **Intelligent Dispatch**: Context-aware volunteer matching.
- **Interactive Heatmaps**: Visualizing active crises and risk zones.
